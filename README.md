# WAI-ARIA-Accordion [(Live example)](http://wai-aria-accordion.surge.sh/)
An WAI-ARIA Accordion component which can be handled by keyboard.

* ### Keyboard Actions:
	* → | ↓ = Moves the tab down
	* ← | ↑ = Moves the tab up
	* Enter | Spacebar = Togles tab displays or hides tab content.
